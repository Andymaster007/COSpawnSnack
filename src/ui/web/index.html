使用须知

它是干什么的

自动打开指定的网页并在复活后切回游戏。（刷视频支持自动播放和暂停上一个在看的视频）

电脑需要什么

系统：Windows 10（2020 年以后的版本）或 Windows 11。更老的 Windows 用不了。
浏览器：电脑上装了 Chrome 或 Edge 其中一个就行（在「配置」里选）。
中文识别：如果你的电脑系统已经是简体中文，中文识别一般已经自带，不用额外装。万一出现"明明死了却没切出去"，多半是没装中文识别——去「设置 → 时间和语言 → 语言 → 中文(简体) → 选项」里看看有没有"光学字符识别"，没有就点"添加"。
游戏里：窗口模式建议选择为全屏窗口。

怎么用

① 在「配置」选好浏览器和要打开的网站（也能填自己的网址）；② 回「控制台」点「开始监控」；③ 正常打游戏；④ 想停就再点一次，或直接关窗口。
热键：F8 或 Ctrl+F8 可随时开始 / 停止 监控（若所选浏览器未安装，程序会自动改用其他可用的浏览器）。

测试一下

点下面的「测试切换」按钮，可手动触发一次"切出去 / 切回来"，确认浏览器和网址设置没问题。
