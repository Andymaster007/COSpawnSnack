# CODMSpawnSnack · CO摸鱼管理器（COSpawnSnack）

> 🌐 [中文](README.md) · [English](README.en.md)

> 一个 Windows 小工具：我一死就自动切到设定好的网页，回合结束再自动切回来。

## 想法来源

经常和朋友打游戏，他阵亡后就去刷视频，结果刷得太投入，忘了切回来，导致在对局中被踢了出去。

既然观战时间总有人忍不住摸鱼，那不如让电脑来管理摸鱼时间：**该摸鱼时自动切过去，该回来时自动切回来**，既不耽误刷视频，也不会因为忘了切屏被踢。CO摸鱼管理器就是这么来的。

## 它怎么工作

整个过程不读内存、不注入游戏，只靠“看画面”：

1. **抓画面**：用 **Windows Graphics Capture (WGC)** 在后台捕获游戏窗口（旧机器上自动回退到 BitBlt）。
2. **认死亡**：对屏幕中下方的复活提示条做 **Windows.Media.Ocr** 识别，读到“你将在下一回合重生”即判定为阵亡。
3. **认结束**：对屏幕中央做 OCR，读到“胜利 / 战败”即判定回合或对局结束。
4. **切屏 + 续播**：阵亡时切到短视频并继续播放，复活 / 对局结束时切回游戏并把视频暂停在原进度。暂停 / 继续通过系统媒体控制 (GSMTC) 读取真实播放状态后精确操作，不会和你手动暂停“打架”。

## 主要特性

- **纯视觉检测**：只做外部画面识别，不读取、不修改游戏内存，规避注入类反作弊风险。
- **视频不丢进度**：切回游戏时视频只是暂停，回来刷还是同一条、同一进度。
- **浏览器自适应**：优先使用你选择的浏览器，未安装时自动回退到其他已安装的适配浏览器（Chrome 优先，其次 Edge）。
- **双热键**：`F8` 随时开始 / 停止监控；若 `F8` 被其他程序占用，可用 `Ctrl+F8` 备选。
- **启动自检**：启动时检测 Windows 版本与可用浏览器，不满足会给出中文提示。
- **单文件 exe**：静态链接，内嵌 WebView2 界面，双击即用。

## 技术栈

C++20 · MSVC · CMake · vcpkg · OpenCV · Win32 · Windows Graphics Capture · Windows.Media.Ocr · WebView2

## 运行环境

- Windows 10 1803（内部版本 17134）及以上 / Windows 11。
- 需安装 Microsoft Edge WebView2 Runtime（Win11 及较新 Win10 通常已自带）。
- 暂只支持 Chrome 或 Edge 浏览器。

## 使用须知

- 检测机制基于 OCR，请务必将**局内按键透明度调至 100%**，否则可能识别失败。
- 目前**仅支持爆破模式**，使命战场（大逃杀）不在本版本范围内。
- 按 `F8` 或 `Ctrl+F8` 开始 / 停止监控。
